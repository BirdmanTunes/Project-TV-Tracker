Welcome to TV SHOWS TRACKER, an app designed to make your tracking of shows much easier.

Features include:

- Two amazing Light and Dark themes
- Multiple lists easily transferable between each other
- Commiting when clicking outside of cell
- Add button that only activates when the first 3 fields are not empty
- News tab that lets you know when a new episode is out based on the date 
  you set in Season Ended(Date format must always be the same)
- Save button that saves all tabs simultaneously
